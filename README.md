# 🍽️ TasteCraft

A modern and user-friendly recipe web application built using **React** and **Material UI**, designed to provide a seamless cooking and browsing experience.

---

## 🚀 Live Demo

🔗 https://your-vercel-link.vercel.app

---

## 📌 Features

* 🔍 Browse and explore a variety of recipes
* ❤️ Add and manage favorite recipes
* 🔐 Login & Registration UI with validation
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🎨 Clean and modern UI using Material UI
* 💾 Data persistence using localStorage

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **UI Library:** Material UI (MUI)
* **Routing:** React Router
* **State Management:** useState, useEffect
* **Deployment:** Vercel
